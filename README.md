# Patrik Marius

Root workspace checkout for `marius-patrik`.

This repository tracks selected repositories as top-level Git submodules. The home directory uses a default-deny `.gitignore`, so local files outside the workspace manifest are not staged by default.

## Submodules

- `Agents` - wrapper repository containing `Andromeda` as a submodule.
- `Archive` - archived repository pointers, currently including `Wrkspace`.
- `Research` - research documents and proof artifacts.
- `TailStreamer` - media streamer project.
- `Fabrica` - private project.
- `RSCode` - public project.
- `Private` - private project.

`Blog` is not listed yet because it has no branch commit to pin as a submodule.
